## Image Guidelines

- **File format:** JPG for photos, PNG for graphics with transparency
- **Optimization:** Compress images to reduce file size without losing quality
- **Quality:** Use high-resolution images that look sharp on retina displays
- **Consistency:** Maintain consistent styling across project thumbnails
- **File naming:** Use lowercase with hyphens (e.g., `project-name.jpg`)

## How to Add Images

1. Place your images in this `assets` folder
2. Name them exactly as listed above, or update the image paths in `index.html`
3. Ensure images are optimized for web (compressed)
4. Test the website to verify all images load correctly
